# DMS Contract - 印刷全流程计划书与合同生成系统

一个在线可编辑的方案配置与合同生成系统，专为印刷行业设计。

## 功能特点

### 方案配置
- **方案名称可编辑**：点击表头的方案名称或副标题可直接修改
- **动态添加/删除方案**：支持灵活增减方案列
- **勾选功能**：点击单元格可在 ✔ / ✘ / 空白 之间切换
- **价格可编辑**：点击价格单元格可直接输入价格

### 服务级别管理
- **服务级别可编辑**：点击服务级别表格的单元格可直接编辑内容
- **动态添加/删除服务级别**：支持灵活增减服务级别

### 合同生成
- **一键生成合同**：选择方案后自动生成完整合同
- **合同预览**：实时预览合同内容
- **打印/下载**：支持打印和下载合同（HTML格式）

### 数据管理
- **自动保存**：所有修改自动保存到浏览器本地存储
- **导出Excel**：支持导出方案配置为CSV文件
- **重置功能**：一键重置所有数据

## 使用方法

1. 打开 `index.html` 文件
2. 在"方案配置"页面配置各方案的功能勾选和服务级别
3. 点击"下一步"进入"合同信息"页面
4. 选择一个方案，填写甲乙双方信息和价格
5. 点击"生成合同预览"查看完整合同
6. 可打印或下载合同

## 技术栈

- 纯HTML/CSS/JavaScript
- 无需后端服务器
- 数据存储在浏览器localStorage
